# Instruktioner for GitHub
Initiering:

git init

git remote add Project https://github.com/FJ6855/TSBK07_Project.git

Ladda ner filer:

git pull Project master

När ändringar gjorts:

git add .

git commit -m "message"

Se status av filer:

git status

Ladda upp filer:

git push
